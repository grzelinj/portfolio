Obsidian Second Brain

personal project · runs locally

A personal knowledge-management system driven by conversation with Claude Code — automatically transcribes, categorizes and links everything I capture by voice, from a web clipper, or from YouTube/Instagram, then emails a weekly summary.

Expand ↓

Three intake channels (Whisper Flow for voice notes, a web clipper for articles, "share to Obsidian" from YouTube/Instagram) all feed into one Inbox. transcribe.py turns video links into text (free YouTube captions, or local Whisper AI for Instagram), and saying "review the inbox" launches Claude Code, which categorizes, tags and links everything into the right knowledge folders. Claude Code also runs a Cron job that nags for a weekly review, after which send_review_email.py emails a "Morning Press" summary.

Python Claude Code (custom skills + Cron) Whisper AI youtube-transcript-api Obsidian Gmail SMTP

Runs entirely locally — no public demo (touches private notes and email).
