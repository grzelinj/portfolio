Asystent diety

z Kacprem Rogowskim

Aplikacja Streamlit, która na podstawie profilu i celu generuje spersonalizowaną rekomendację diety i plan posiłków przez Claude API.

Rozwiń ↓

Wspólny projekt z Kacprem Rogowskim. Na podstawie profilu, celu i preferencji Claude generuje pełną rekomendację diety (makro, produkty, suplementacja) oraz plan posiłków na cały tydzień, z eksportem do PDF. Produkty można też zeskanować kodem kreskowym — dane pobierane z Open Food Facts.

Streamlit Python Claude API Open Food Facts API
▶ Zobacz demo ↗ github.com/grzelinj/asystent-diety-demo

Demo z przykładowymi danymi (bez realnych zapytań do Claude API) — pierwsze wejście może potrwać ~30–50s, bo darmowy hosting usypia po bezczynności.

Obsidian Second Brain

projekt osobisty · działa lokalnie

System zarządzania wiedzą osobistą sterowany rozmową z Claude Code — automatycznie transkrybuje, kategoryzuje i linkuje wszystko, co przechwycę głosem, z web clippera albo z YouTube/Instagrama, a raz w tygodniu wysyła mailowe podsumowanie.

Rozwiń ↓

Trzy kanały wejścia (Whisper Flow do notatek głosowych, web clipper na artykuły, "share to Obsidian" z YouTube/Instagrama) trafiają do jednego Inbox-a. transcribe.py zamienia linki wideo na tekst (darmowe napisy YouTube albo lokalny Whisper AI dla Instagrama), a komenda "przejrzyj inbox" uruchamia Claude Code, który kategoryzuje, taguje i linkuje wszystko do właściwych folderów wiedzy. W Claude Code działa też Cron przypominający o cotygodniowym przeglądzie, po którym send_review_email.py wysyła podsumowanie "Poranna Prasa" na maila.

Python Claude Code (custom skills + Cron) Whisper AI youtube-transcript-api Obsidian Gmail SMTP

Działa wyłącznie lokalnie — brak publicznego demo (dostęp do prywatnych notatek i poczty).
